# Campus Hustler

A platform to help college students find short-term jobs.

## Features
Post short-term jobs
Find jobs by seeing job listings
Students can pick up a job from the listing

## Creation
using the job board template from GitHub
using Python and Flask

## Setup Instructions

### Create a virtual environment
```bash
python -m venv .\venv
```

### Activate the virtual environment

```bash
.\venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run tests

```bash
pytest
```

### Run the app

```bash
flask run
```

Then open your browser and go to:

```text
http://localhost:5000
```
